ALL BASIC HTML TAGS:



1. html: Root Element
Definition: The container for the entire HTML document.

2. head: Head Section
Definition: Contains metadata and links to stylesheets or scripts.

3. title: Page Title
Definition: Sets the title displayed on the browser tab.

4. body: Body Section
Definition: Contains all visible page content.

5. h1 to h6: Headings
Definition: Define headings, with h1 as the largest and h6 the smallest.

6. p: Paragraph
Definition: Defines a block of text as a paragraph.

7. a: Anchor (Link)
Definition: Creates a hyperlink to another page or section.8. : Image
Definition: Embeds an image using the src attribute.

9. ul, ol, li: Lists
Definition:

ul creates an unordered list (bulleted).
ol creates an ordered list (numbered).
li defines list items.

10. table, tr,td: Table Elements
Definition:

table defines a table.
tr defines a row.
td defines a cell (data).

11. form: Form for User Input
Definition: Creates a form for collecting input from users.
